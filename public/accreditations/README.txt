ACCREDITATION / CERTIFICATION LOGOS
===================================

These are the brand logos shown in the "Certified & Accredited" strip
(component: src/components/Accreditations.tsx).

The files here are PLACEHOLDER wordmarks. Replace each with the official
brand logo, keeping the SAME filename so no code changes are needed:

  magnum.svg    -> Magnum Ceramic Coating (main / graphene coating)
  kraken.svg    -> Kraken (self-healing coating)
  gtechniq.svg  -> Gtechniq Certified / Accredited Detailer
  carpro.svg    -> CarPro

You can drop in .png instead of .svg, but then update the extension in
src/components/Accreditations.tsx (the `${b.slug}.svg` line).

Tip: use logos with a transparent background — they sit inside white chips.
